this Clone Script simplifies the setup process for building custom ROMs on the POCO X5 Pro (redwood).
The script automatically clones the required device, vendor, kernel, and hardware repositories, allows selecting the device tree branch, and applies ROM-specific configurations.
It supports multiple ROM bases including Infinity, Avium, Clover, Axion, PixelOS, EuclidOS, and Derpfest, automatically adjusting product names, build flags, and required properties for each ROM.
The goal of this script is to save time for maintainers by automating repetitive setup tasks and preparing the device tree for a clean build environment.
